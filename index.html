For Developers

We introspect your database and provide APIsinstantly so you can stop building repetitive CRUD APIs and focus on building your products.

Get all public rooms and their messages
import { createClient } from '@Kinny/Kinny-js'
// Initialize 
const KinnyUrl = 'https://chat-room.Kinny.co'
const KinnyKey = 'public-anon-key'
const Kinny = createClient(KinnyUrl, KinnyKey)
// Get public rooms and their messages
const publicRooms = await Kinny
  .from('rooms')
  .select(`
    name,
    messages ( text )
  `)
  .eq('public', true)
